# Project Overview

This project was created to simulate a complete manual QA testing workflow using Jira and Zephyr.

## Application
AgileWay Travel (demo web application)

## Objective
The main goal of this project is to demonstrate:
- Manual test case design
- Test execution using test cycles
- Bug reporting with clear and structured information
- Organization of test artifacts by feature

## Testing Approach
- Manual functional testing
- Positive and negative scenarios
- Validation of user flows and business rules

## Tools
- Jira Cloud
- Zephyr Scale
